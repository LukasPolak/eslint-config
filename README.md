# `@lukaspolak/eslint-config`

> My personal [eslint](https://eslint.io) config.

## Usage

**Install**:

```bash
yarn add @lukaspolak/eslint-config --dev
```

or

```bash
npm i @lukaspolak/eslint-config --dev
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "eslint": "@lukaspolak/eslint-config"
}
```
